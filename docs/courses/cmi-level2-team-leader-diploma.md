## Unit Detail: 2001V1 – Personal Development as a Team Leader

### Learning Outcomes
1. Be able to identify work objectives and the skills and knowledge required to meet them
2. Be able to agree, implement and review the development plan to meet work objectives
3. Understand team leader's responsibility for welfare in the work area
4. Understand the team leader's responsibilities in relation to discrimination, harassment and diversity

### Assessment Tasks (Written Assignments)

| Task | Focus | Assessment Criteria | Guideline Word Count |
|------|-------|---------------------|---------------------|
| Task 1 | Identify main work objectives, explain how they are set and agreed | AC 1.1 | 150–200 |
| Task 2 | Identify knowledge/skills for a team leader, compare against own, construct PDP | AC 1.2, 1.3, 1.4, 1.5, 1.6 | 350–450 |
| Task 3 | Explain how to agree, implement and review PDP | AC 2.1, 2.2, 2.3 | 450–550 |
| Task 4 | Team welfare responsibilities, positive/negative factors, adapting practices | AC 3.1, 3.2, 3.3, 3.4 | 450–550 |
| Task 5 | Discrimination, harassment, diversity and inclusion in the team | AC 4.1, 4.2, 4.3, 4.4 | 150–250 |

### Marking
- External marking by CMI (Pass / Referred)
- CMI Lead Moderator review
- Resubmissions: 1st referral in blue, 2nd in red to differentiate

## Learning Resources (Per Unit)

Resources are downloadable tools/templates that support learners in completing their assignments. They are NOT teaching materials — learners use them independently.

### Self-Awareness & Development Tools

| Resource | Type | Purpose |
|----------|------|---------|
| SWOT Analysis Sheet | Template | Evaluate current and future professional development needs (Strengths, Weaknesses, Opportunities, Threats) |
| Personal Development Plan (PDP) | Template | Outline specific actions, start/target dates, outcomes and impact over 12 months |
| CPD Log | Template | Record CPD activities with date, description, learning outcome, and workplace impact |
| Honey & Mumford Learning Styles | Guidance + Questionnaire | Identify learning preference (Activist, Theorist, Pragmatist, Reflector) |
| VARK Questionnaire | Questionnaire | Identify learning style preference (Visual, Auditory, Read/Write, Kinesthetic) — 16 questions with scoring grid |
| Johari's Window | Guidance + Activity | Self-awareness exercise — 57 adjectives mapped to Open/Blind/Hidden/Unknown quadrants |
| 360 Degree Feedback Form | Template | Collect behavioural feedback from peers across 6 competency areas (Self-Awareness, Drives for Results, Communication, Collaboration, Developing Others, Problem Solving) with frequency rating scale |
| Emotional Intelligence Questionnaire | Questionnaire (Excel) | Boston EI questionnaire — self-assessment across 5 EI dimensions with auto-scored results |

## Completion Workflow

1. Access learning resources for each unit
2. Complete self-assessment tools (SWOT, VARK, Honey & Mumford, Johari, EI, 360 Feedback)
3. Complete Knowledge Check Quiz (≥80%)
4. Write and submit assignments per unit (Tasks 1–5 for Unit 2001, etc.)
5. CMI external marking → Pass or Referred
6. If Referred → resubmit in blue (1st) / red (2nd)
7. All units passed → Qualification awarded

## Platform Implications

This course structure differs from the Common Core course and informs how the platform must handle **both** patterns:

1. **Resource types**: Includes interactive questionnaires (VARK, EI, Honey & Mumford) not just static PDFs — platform needs fillable template support or downloadable Excel/DOCX
2. **Assignment-based assessment**: Written tasks assessed against specific Assessment Criteria codes (AC 1.1, 1.2, etc.) — NOT module-by-module completion like Common Core
3. **External marking**: CMI marks externally — platform must support export/packaging of submissions for external review
4. **Resubmission tracking**: Must track 1st vs 2nd referral distinctly
5. **Quiz as pre-check**: Quiz validates readiness BEFORE assignment submission (different from Common Core where quiz IS the assessment)
6. **Mixed question formats**: MCQ + short-answer questions in the same quiz
7. **Self-assessment tools**: Non-graded activities that feed into assignment evidence (e.g., completed SWOT → referenced in PDP assignment)
